# ViewFinder

Head over to [https://sharadbhat.github.io/viewfinder](https://sharadbhat.github.io/viewfinder) to check it out!

## To run locally

```sh
# Clone repo
git clone https://github.com/sharadbhat/viewfinder.git

# Install packages
npm i

# Run
npm run start
```
